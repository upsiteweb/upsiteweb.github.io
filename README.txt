UpSite — Site estático (HTML + CSS + JS)
=========================================

Estrutura:
  index.html       — página principal
  styles.css       — estilos
  script.js        — interações
  assets/          — imagens do portfólio

Como usar:
  1) Abra index.html direto no navegador, OU
  2) Faça upload de toda a pasta para qualquer hospedagem
     (Netlify, Vercel, Hostinger, GitHub Pages, etc.)

WhatsApp:
  Substitua o número 5551999999999 em index.html pelo seu.

Instagram:
  Atualize o link @upsite.web em index.html.
